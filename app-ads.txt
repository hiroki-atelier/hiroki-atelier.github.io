# app-ads.txt
#
# このファイルは、広告（AdMobなど）を導入する際に必要になります。
# 現時点ではプレースホルダです。広告未導入の間は中身が空でも問題ありません。
#
# AdMobを導入したら：
#  1. AdMob管理画面 → 「アプリ」→「app-ads.txt」セクションを開く
#  2. AdMobが提示する1行（下記のような形式）をコピー
#  3. このファイルの中身を、その行だけに置き換える（このコメント行は削除してよい）
#
# 例（※これはサンプルです。必ずAdMobが提示する自分の行を使うこと）：
# google.com, pub-0000000000000000, DIRECT, f08c47fec0942fa0
